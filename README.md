# YouCanApp

Plataforma educativa moderna enfocada en tutorías personalizadas, webinars interactivos en vivo, asesorías grupales, clases grabadas y recursos académicos.

## Características principales

- **Tutorías personalizadas** — match automático con el tutor ideal usando IA
- **Webinars en vivo** — salas interactivas tipo Teams/Zoom con chat, reacciones y pantalla compartida
- **Asesorías grupales** — sesiones colaborativas de hasta 5 participantes
- **Clases grabadas** — acceso a grabaciones con transcripción y resumen IA
- **Calendario académico** — sincronización automática de todas las sesiones
- **Recursos académicos** — materiales, quizzes interactivos y notas personales

## Stack tecnológico

- Frontend: React + TypeScript
- Estilos: Tailwind CSS
- Backend: Node.js
- Base de datos: PostgreSQL

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/stolasblitzo086-ai/youcanapp.git
cd youcanapp

# Instalar dependencias
npm install

# Iniciar en desarrollo
npm run dev
```
